# Rotate2d
his project implements a 2D Rotation Engine in Verilog HDL that rotates a point (x, y) by 0°, 90°, 180°, and 270° using signed arithmetic. The design is fully verified using a Verilog testbench in ModelSim.  Features Supports signed 8-bit input coordinates. Performs four standard 2D rotations: 0°: (x, y) 90° : (-y, x) 180° : (-x, -y) 270° : (y, -x)
